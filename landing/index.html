Common questions

Will this work with Tally.ERP 9 or only TallyPrime?

TallyWhatsApp is built for TallyPrime (4.x and newer). Tally.ERP 9 uses an older TDL contract and is not officially supported. If you're on TallyPrime, you're good.

Will my WhatsApp account get banned?

We use the official WhatsApp Multi-Device protocol — the same one WhatsApp Web uses. Your account is treated exactly like a linked device, and we don't automate beyond the protocol, scrape contacts, or bypass any policy. That said, WhatsApp's own anti-spam system can flag any account — yours, ours, or a manual user's — that sends a large bulk burst. If you blast a few hundred invoices back-to-back, WhatsApp may temporarily restrict your account for around 24 hours. We pace receipt vouchers automatically (~90s apart) for this reason. For sales batches, pace sends to real customers who expect them — that's all the spam filter cares about.

What if I have multiple companies in Tally?

Multi-company is fully supported. The Send WhatsApp button and auto-prompt appear in every company file Tally opens — no extra setup per company.

Can I move TallyWhatsApp to a new computer?

Yes. Each license includes 3 device activations. Install on the new PC, paste the same key, and it works. If you ever run out, email admin@variantstudio.in and we'll reset.

Will my customers' phone numbers leak to a third party?

No. The bridge runs entirely on your PC. WhatsApp messages go directly from your machine via the official Multi-Device protocol — there's no relay server, no cloud database, no external API in the loop.

What happens when TallyPrime updates?

The connector self-heals on each service restart and re-patches Tally's tally.ini if needed. The underlying TDL contract is stable, so Tally updates rarely break us — and if they ever do, we ship an update within days.

Why is the QR stuck on "Preparing"?

Open PowerShell as Administrator and run Restart-Service TallyWhatsAppConnector. Refresh the dashboard — a fresh QR appears within a few seconds. This usually only happens after a Windows sleep/wake or a forced shutdown.
